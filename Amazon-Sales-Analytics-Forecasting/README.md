# 🛒 Amazon Sales Analytics & Forecasting
Python • SQL • Power BI • Time-Series Modeling

## Project Overview  
An end-to-end analytics project designed to analyze Amazon sales performance, profitability, customer trends, and forecast future demand using time-series models.

## Architecture  
Python Cleaning → SQL Data Modeling → Power BI → DAX KPIs → Forecasting (ARIMA/Prophet)

## Key Features  
- Data cleaning using Python  
- SQL fact–dimension model  
- Power BI dashboard  
- Forecasting (30 days)  
- Anomaly detection  
- Automated refresh-ready pipeline

## Dashboard Pages  
- Sales Overview  
- Category Analysis  
- Profitability  
- Region Insights  
- Forecasting  
- Anomaly Detection
