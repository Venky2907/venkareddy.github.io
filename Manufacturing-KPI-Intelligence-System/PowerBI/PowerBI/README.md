# Power BI Dashboard – Manufacturing KPI Intelligence System

This folder represents the Power BI layer of the Manufacturing KPI Intelligence System.

## Dashboard Overview
The Power BI dashboard provides a unified view of plant-level KPIs across:

- Safety
- Quality
- Energy Management
- Water Management
- Production & Maintenance
- Supply Chain & Sales

## Key Capabilities
- Interactive KPI dashboards
- Drill-down and drill-through analysis
- Department-wise and product-wise views
- Trend analysis for operational monitoring
- Used by leadership teams for daily performance review

## Data Sources
- SQL Server (cleaned and validated data)
- SharePoint
- Excel files
- CSV files

## Refresh Strategy
- Automated dataset refresh using Power BI On-Premises Gateway
- No manual intervention required

> Note:  
> The actual `.pbix` file is not uploaded to GitHub to avoid sharing internal business logic and data models.
